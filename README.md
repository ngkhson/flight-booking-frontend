# ✈️ StingAir - Flight Booking Frontend


## 📌 Overview

StingAir is a modern flight booking frontend application that provides users with an intuitive interface to search flights, view flight details, and manage bookings.

The project focuses on responsive UI, reusable components, and modern frontend development practices.


## ✨ Features

### 👤 User

- View available flights
- Search and filter flights
- View flight details
- Select seats
- Booking interface
- User profile page


### 🎨 UI/UX

- Responsive design
- Modern user interface
- Reusable components
- Smooth navigation


## 🛠️ Technologies

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

Libraries:

- React Router
- Axios
- Bootstrap / Tailwind CSS
